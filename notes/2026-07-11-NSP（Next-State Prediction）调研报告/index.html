NSP（Next-State Prediction）调研报告

生成日期：2026-07-11

说明：NSP（Next-State Prediction）并非具体软件产品，而是由智源研究院提出的 AI 新范式。本报告将其作为独立概念进行系统调研。

1. 概念提出者与背景

1.1 提出机构

北京智源人工智能研究院（BAAI），成立于 2018 年，由中国科技部和北京市政府联合支持，是国家级 AI 研究机构。

智源已在多模态世界模型领域取得一系列标志性成果：悟道系列大模型、Emu 系列多模态模型、FlagScale 训练框架等。2024 年，Emu3 成为全球首个原生多模态世界模型；2025 年，Emu3.5 登上 《Nature》正刊，是国内多模态大模型首篇。

1.2 核心人物

王仲远——智源研究院院长，NSP 范式的主要提出者和推动者。

  • 曾任 Facebook、谷歌、微软等公司 AI 研究负责人
  • 2026 年 1 月智源大会上正式提出 NSP 范式
  • 核心论断：“基础模型的竞争，焦点已从’参数有多大’转变为’能否理解世界如何运转‘“

1.3 合作机构与生态

类型机构
学术合作清华大学、北京大学、中科院等
产业化智源悟界系列模型已在具身智能、自动驾驶等领域落地测试
开源生态悟界·Emu3、Emu3.5、Physis 等模型部分开源

信息源：新浪科技专访王仲远 · 央视网 智源2026十大趋势 · 新华网报道 · 智源官网

2. 底层技术解析

2.1 什么是 NSP？

NSP（Next-State Prediction） 是一种全新的 AI 学习范式——模型不再预测”下一个词是什么”，而是预测”世界的下一个物理状态是什么”。

2.2 三代范式的本质区别

NTP（Next-Token Prediction）         GPT 系列、LLaMA
   │  预测：下一个词
   │  理解：语言统计规律
   │  盲区：不懂物理世界
   ▼

NPF（Next-Pixel/Frame Prediction）    Sora、视频模型
   │  预测：下一帧像素
   │  理解：视觉表面连续性
   │  盲区：像素变化≠物理理解
   ▼

NSP（Next-State Prediction）⭐⭐⭐    智源 Emu3.5、悟界·Orca
   │  预测：下一个物理状态
   │  理解：物理因果规律
   │  突破：从"看起来像"到"真的懂"

2.3 技术实现路径

NSP 的核心是一套统一物理潜空间表征 + 状态转移建模框架：

多模态输入（文本/图像/视频/动作）
         │
         ▼  离散化为统一 Token 序列
┌─────────────────────────────────┐
│       统一 Token 化空间          │
│  "杯子" = token_id_1423         │
│  杯子图像 = [token_id_1423, ...]│
│  杯子掉落视频 = [token_seq]     │
└──────────┬──────────────────────┘
           │  Self-Attention + State Transition
           ▼
┌─────────────────────────────────┐
│    状态转移学习（State Dynamics） │
│  S₁（杯在桌沿）                 │
│    → 重力作用                   │
│  S₂（杯开始倾斜）               │
│    → 力矩计算                   │
│  S₃（杯脱离桌面）               │
│    → 自由落体                   │
└──────────┬──────────────────────┘
           ▼
┌─────────────────────────────────┐
│    输出：下一个物理状态预测       │
│  文本描述 / 视频帧 / 动作指令    │
└─────────────────────────────────┘

2.4 智源悟界·Emu3.5 技术架构

维度规格
参数量34B（340 亿）
训练 Token超 10 万亿（两阶段预训练约 13 万亿）
视频数据6,300 万条长视频，总时长 790 年（仅占全网公开视频不到 1%）
数据预处理Whisper ASR + PySceneDetect 关键帧提取 + 图文交错序列
训练流程两阶段预训练 → SFT（1,500 亿样本）→ 大规模强化学习 → DiDA 快速适配

2.5 三大核心技术创新

1️⃣ 大规模多模态预训练

  • 文本、图像、视频、动作全部离散化为统一 Token 序列
  • 模型不再区分模态——“一张图”和”一句话”用同一套”世界积木”表示
  • 两阶段训练：基础预训练（10万亿 Token）→ 高分辨率/高质量微调

2️⃣ 大规模原生多模态强化学习（RL）

  • 多模态领域首次在自回归架构下落地大规模 RL
  • 构建综合奖励系统：通用性 + 任务特异性 + 统一性
  • 多维奖励避免单目标过拟合

3️⃣ DiDA（离散扩散自适应）推理加速

  • 解决自回归架构图像生成慢的瓶颈
  • 将离散扩散公式扩展到视觉 Token
  • 单图推理速度提升近 20 倍，媲美顶尖闭源扩散模型

4. 功能全景解析

核心能力（一句话概括）

NSP 让 AI 从”续写文字”升级为”理解世界因果”——输入多模态信息，输出对下一个物理状态的预测与推理。

4.1 🧠 物理世界理解

维度说明
物体理解识别物体身份、位置、姿态、材质属性
关系推理理解物体间的空间关系、因果关系
物理规律内化重力、摩擦、碰撞、流体等基本物理规律

王仲远经典举例：

一瓶未开封的水和盛满咖啡的杯子放在桌旁。人类能预测：水瓶跌落只有撞击声，咖啡杯会洒落一地。这就是世界模型。传统 AI 缺乏这种”下一个物理状态”的预测能力。

4.2 🔮 下一个状态预测

输入输出示例
文本描述物理状态描述”杯子在桌沿”→“即将掉落”
图像下一时刻图像静态场景→预测物体运动趋势
视频延续视频前 5 秒→预测后 5 秒的物理演变
动作指令动作结果”把杯子拿起来”→预测手的位置变化

4.3 🎯 意图到规划

  • 理解高层级人类意图（如”如何做咖啡拉花""如何叠衣服”）
  • 自主生成详细、连贯的多步骤行动路径
  • 在统一框架内无缝融合理解、规划与模拟

4.4 🔄 动态世界模拟

  • 预测物理动态、时空演化和长时程因果关系
  • 支持反事实推理（Counterfactual Reasoning）：“如果不这样做会怎样？”
  • 为具身操控提供认知基础

4.5 🏗️ 第三个 Scaling 范式

王仲远提出：NSP 开启了继”语言预训练 Scaling”和”后训练/推理 Scaling”之后的第三个 Scaling 范式：

Scaling 范式代表核心特征
第一代GPT 系列语言预训练参数规模扩展
第二代o1、DeepSeek-R1后训练 + 推理时间扩展
第三代（NSP）智源 Emu3.5多模态大一统 + 大规模 RL + 基础设施复用

信息源：InfoQ 王仲远专访 · 极客公园 第三个 Scaling 范式 · 搜狐 Emu3.5 NSP 技术

5. 功能实现案例

案例：悟界·RoboBrain Orca —— NSP 驱动的具身大脑

场景：机器人需要完成”从桌上拿起一个易碎的高脚杯，放到洗碗机上层支架”的任务。

传统 VLA 方案：

输入：视觉画面 → 直接输出机械臂关节角度
问题：每种杯子、每种摆放位置都需要大量真实数据训练
      遇到没见过的杯子或布局，泛化能力差

NSP 方案（悟界·Orca）：

输入：7 路视觉画面 + 机械臂状态
    ↓
Step 1：物理状态抽象
   └─ 将场景抽象为统一潜空间表征：
      高脚杯（易碎材质，重心偏高，杯柄脆弱）
      桌面高度 0.75m，洗碗机上层高度 0.4m
      当前抓取角度 30°
    ↓
Step 2：NSP 状态推演
   └─ 在潜空间中预演多条路径的物理结果：
      ├─ 路径 A：直接抓杯口 → 预测→ 杯柄应力超标 → 可能断裂 ❌
      ├─ 路径 B：抓杯柄底部 → 预测→ 应力安全 → 继续推演
      │              ↓ 放入洗碗机 → 预测→ 高度冲突 → 旋转 15° ✅
      └─ 路径 C：先平移再抓取 → 预测→ 耗时较长但稳定 → 备选
    ↓
Step 3：选择最优路径 → 输出动作序列

结果：NSP 让机器人在没有真实数据的情况下，通过在潜空间中”预演”物理结果来选择最优动作，泛化到从未见过的杯型和布局。

来源：智源大会 2026 现场演示

6. 主要竞品/对比范式

6.1 三代范式对比

#范式代表核心逻辑根本局限
1NTP（Next-Token Prediction）GPT-5、LLaMA 4预测下一个词，学习语言统计规律无物理理解，不懂因果
2NPF（Next-Frame Prediction）Sora 2、Genie 3预测下一帧像素，隐式学习物理像素变化≠物理理解，长程一致性差
3NSP（Next-State Prediction）智源 Emu3.5、悟界·Orca预测下一个物理状态，显式建模因果⭐⭐⭐ 本报告主题

6.2 世界模型四大分类（智源分类）

类别代表距离真正的 NSP
以语言为中心（VLM、VLA）GPT-4V、LLaVA❌ 学到的是语言描述的世界
以像素为中心（视频生成）Sora、Cosmos-Predict❌ 更像模拟而非状态预测
以 3D 结构为中心（重建）NeRF、3DGS❌ 几何结构≠物理状态
以视觉表征为轴心（JEPA）V-JEPA、AMI Labs⚠️ 视觉演化≠物理规律
以 NSP 为核心（智源路线）Emu3.5、Physis✅ 目标直达物理状态预测

6.3 产业落地代表

产品领域与 NSP 的关系
蔚来 NWM 2.0自动驾驶隐式 NSP——预测未来 3 秒物理状态
小鹏 X-World自动驾驶仿真隐式 NSP——日仿真 3000 万公里
悟界·RoboBrain Orca具身智能显式 NSP——直接预测下一个物理状态
NVIDIA Cosmos 3物理 AI分层架构，Reasoner 塔做状态推演

7. 横向比较

7.1 三代范式多维对比

对比维度NTP（Next-Token）NPF（Next-Frame）NSP（Next-State）
预测单位单个词/子词整帧像素阵列结构化状态向量（位置/速度/力/关系）
训练数据文本语料视频帧序列多模态序列（视频/传感/物理仿真）
学习目标语言概率分布像素级视觉连续性状态转移概率分布
核心能力语言流畅性视觉时间连贯性因果物理理解
物理理解❌ 完全不懂⚠️ 隐式、不可靠✅ 显式因果建模
可否规划❌ 不能⚠️ 有限✅ 可预演多路径
算力需求中极高（像素级生成）高（但潜空间远低于像素）
长程一致性✅ 文本可很长❌ 误差指数累积⚠️ 潜空间好于像素但仍有挑战
代表性基准MMLU、HumanEvalVBench、Physics-IQWorldModelBench
最适用场景对话、写作、编程视频生成、创意内容机器人、自动驾驶、科学模拟

7.2 各范式最适用场景

范式最佳场景不应使用的场景

一句话锐评

NSP 是 2026 年最重要的 AI 范式宣言——它宣告了”预测下一个词”的时代正在让位于”预测世界如何运转”。智源用 Emu3.5 证明了这条路的可行性，但真正的大考在于：这种物理因果建模能否像 NTP 一样，在规模扩展中持续涌现出意想不到的智能。如果答案是肯定的，NSP 将不仅是范式，而是通往 AGI 的路线图。
