坚果云（Nutstore）调研报告

国内专业的文件同步与协作云盘服务商

1. 开发者与公司背景

开发方全称

上海亦存网络科技有限公司（品牌名：坚果云 / Nutstore），注册于中国（上海）自由贸易试验区。

创始人基本信息

姓名角色教育背景职业经历
韩竹创始人、CEO上海交通大学硕士前 IBM 研究院研究员，从事企业级数据中心基础研究
杨剑联合创始人上海交通大学硕士技术极客，负责产品技术研发
蔺皓联合创始人、董事北京邮电大学计算机学院硕士前 IBM 中国云存储研究院，现领导坚果云销售团队

创业故事：2010 年下半年，韩竹受 Dropbox 启发决定创业，联合上海交大好友杨剑和曾是他 IBM 实习生的蔺皓，三人分处北京上海两地，通过邮件和 Skype 协作开发。创业前半年没有工资，窝在韩竹家中办公。2011 年底完成天使轮融资，投资人看中他们”技术很好，短时间内完成了全平台客户端”。

团队规模

约 200 人（部分来源标注为 28 人登记员工，此为工商年报口径；实际运营团队约 200 人左右）。

盈利情况

  • 服务 超 1000 万用户，10 万+ 企业客户
  • 覆盖超 10 个行业的上万家付费企业客户，包括中石油、中海油、银河证券、阿里巴巴、腾讯、清华大学、哔哩哔哩、LVMH 等
  • 采用”增值服务收费”模式（免费版引流 → 付费版盈利）
  • 获 字节跳动战略投资（2018 年起多轮），早期获 纽信创投 天使轮投资
  • 暂无公开的具体营收数据

公司信条与发展规划

  • 信条：“文件随心动，协作零距离”——专注文件同步与协作，不盲目扩张
  • 定位：做中国市场最专业的文件同步与协作平台，而非”大而全”的云存储
  • 发展策略：围绕核心同步能力构建开放生态（WebDAV / API），与工具链深度集成
  • 近期方向：AI 辅助办公、企业工作台、合规安全纵深

2. 底层技术栈

该软件非 AI 原生，不涉及基础大模型。

坚果云是专业的文件同步与协作工具，核心技术栈集中在分布式存储、增量同步算法、网络安全协议等领域，并非 AI 原生软件。

相关底层技术

技术领域具体实现
存储架构自建分布式存储系统，双活数据中心（阿里云 + 腾讯云）
同步算法自研增量同步算法（Block-Level Delta Sync）
加密技术AES-256 存储加密 + SSL/TLS 传输加密
网络优化局域网 P2P 加速、全球 9 个节点加速（新加坡、美欧澳等）
跨平台框架原生 C++ 同步引擎 + 各平台原生 UI
开放协议WebDAV 协议、RESTful API、OAuth2 认证

4. 功能全景解析

主要功能

核心价值：专业、高效、安全的跨平台文件同步与团队协作工具。目标用户：注重文件同步体验的个人知识工作者、科研人员、中小企业团队。

核心功能与关键技术支持

🔑 核心功能一：智能增量同步

  • 技术：自研 Block-Level Delta Sync 算法，将文件拆解为微小的二进制数据块，修改时仅上传发生变动的数据块
  • 效果：修改 200MB PPT 中的一个字符，实际推送仅十几 KB
  • 独特性：国内唯一实现此级别的同步技术，竞品多为全量同步

🔑 核心功能二：WebDAV 协议支持

  • 技术：原生内置 WebDAV 服务器，国内唯一主流支持 WebDAV 的云盘
  • 集成生态：
    • 📚 Zotero：文献附件同步（官方推荐国内方案）
    • 📝 Obsidian / Logseq：笔记库多端同步
    • 📱 Notability / GoodNotes / PDF Expert：手写笔记备份
    • 🗄️ KeePass：密码数据库同步
  • 2025 年升级：推出官方客户端插件（Zotero 同步插件、Obsidian Nutstore Sync），比传统 WebDAV 更稳定、支持大文件、支持增量同步

🔑 核心功能三：全平台覆盖 + 任意文件夹同步

  • 技术：原生 C++ 同步引擎，适配所有主流及国产操作系统
  • 特色：突破”仅同步指定文件夹”限制，可同步电脑中任意位置的文件夹，不改变用户原有文件组织习惯
  • 国产化：适配飞腾、鲲鹏等国产芯片架构

🔑 核心功能四：历史版本管理

  • 技术：差异存储 + 版本链管理
  • 能力：免费版即可保留历史版本，付费版无限版本保存
  • 特色场景：勒索病毒防护——发现中毒后可一键回滚至病毒入侵前状态

🔑 核心功能五：局域网加速

  • 技术：局域网内的设备间自动检测并通过 P2P 直连同步，零公网带宽消耗
  • 适用：办公室多人同步大文件场景

5. 功能实现案例

案例一：Zotero 文献管理同步

场景：研究生小明需要在实验室 Windows 电脑、宿舍 MacBook 和 iPad 之间同步 Zotero 文献库及其 PDF 附件。

使用步骤：

  1. 注册坚果云账号（个人版即可）
  2. 在坚果云网页端「账户信息 → 安全选项」中创建 WebDAV 应用密码（或安装官方 Zotero 同步插件）
  3. 在 Zotero 中设置「首选项 → 同步 → 文件同步」，类型选 WebDAV
  4. 输入坚果云 WebDAV 地址、邮箱和应用密码
  5. 点击「验证服务器」确认连接

效果：文献条目和 PDF 附件在三台设备间自动同步，修改一条笔记后仅几百 KB 增量上传，秒级完成同步。无需手动拷贝、无需 U 盘、版本自动管理。

案例二：Obsidian 知识库多端同步

场景：博主使用 Obsidian 管理个人知识库，需要 iPhone、MacBook 和 Windows 台式机三端无缝同步。

使用步骤：

  1. 订阅坚果云个人版（安装 Nutstore Sync 官方社区插件）
  2. 在 Obsidian 社区插件市场搜索并安装 “Nutstore Sync”
  3. 使用坚果云账号 OAuth 授权登录
  4. 选择要同步的 Vault 文件夹，自动开始同步

效果：Obsidian Vault 在三端保持实时一致，新笔记秒级同步到其他设备。相比传统 WebDAV 方案，官方插件更稳定、支持大文件、且配置仅需一步。

案例三：企业团队图纸协作

场景：某建筑设计团队 10 人，频繁修改大型 CAD 图纸（单文件 100MB+），常出现版本混乱。

使用方案：

  1. 部署坚果云团队版
  2. 设置项目文件夹，分配各成员权限（部分人仅查看、部分人可编辑）
  3. 启用文件锁定功能，防止多人同时编辑同一文件
  4. 硬件出图的版本自动保存历史记录

效果：智能增量同步技术使大型 CAD 图纸每次修改仅上传变动部分（通常几 MB），局域网内同事间直接 P2P 同步零公网消耗。历史版本支持随时回滚，误操作零损失。

坚果云是国内不可替代的「同步型」云盘——增量同步技术和 WebDAV 生态是其护城河，特别适合需要频繁跨设备同步办公文档、科研文献和知识库的专业用户；但免费空间仅 1GB、付费空间容量偏小，不适合海量存储场景。如果你买的是”同步体验”而非”存储空间”，坚果云在同类产品中没有对手。
