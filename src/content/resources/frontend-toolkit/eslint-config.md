## 说明

这是一个适用于 Astro + TypeScript 项目的 ESLint 9 flat config 模板。

### 安装依赖

```bash
npm i -D eslint @eslint/js typescript-eslint eslint-plugin-astro
```

### 配置文件

将 `eslint.config.js` 放在项目根目录即可。详细配置说明见下载文件内的注释。
