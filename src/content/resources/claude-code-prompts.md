## 介绍

这套 Claude Code 提示词模板覆盖了软件开发的 6 个常见场景：

1. **代码审查** — 多维度审查报告，含安全/性能/可读性
2. **重构建议** — 针对特定代码片段的结构化重构方案
3. **Bug 排查** — 系统性排错提示词，从现象到根因
4. **文档生成** — 从代码自动生成 README/API 文档
5. **测试生成** — 基于函数签名生成单元测试
6. **架构分析** — 跨文件依赖分析与优化建议
