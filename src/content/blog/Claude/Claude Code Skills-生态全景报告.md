## 🔑 关键洞察

1. **1,400+ 技能库，真正有用的不到 50** — 推荐安装 8-12 个，超过 30 个后低优先级技能被截断
2. **Superpowers 统治地位** — ~235K Stars，生态绝对标杆
3. **Karpathy 70 行效应** — 仅 70 行 Markdown = 180K Stars，少即是多
4. **安装预算限制** — 每个 Skill 消耗 ~100 token 用于名称/描述扫描
5. **安全风险** — SKILL.md 可含提示注入和凭据窃取，安装前务必审查
