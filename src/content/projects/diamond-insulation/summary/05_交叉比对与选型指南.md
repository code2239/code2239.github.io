## 选型决策树

```
你的核心需求是什么？
│
├─ 最高界面质量 + 量子应用
│   → 选 B（直接键合）
│     亚nm共价键合、单晶膜、NV T₂保体相级
│     代价：成本高、尺寸受限
│
├─ CMOS BEoL兼容 + 功率电子散热
│   → 选 C（低温介电层生长）
│     400°C兼容、标准晶圆级、多平台适用
│     代价：热导率300 W/mK（<单晶2000）
│
├─ 大面积 + 低成本 + Si基散热/传感
│   → 选 A（MPCVD+β-SiC）
│     工业MPCVD、自然界面层、可规模化
│     代价：880°C不兼容BEoL
│
├─ p沟道宽禁带FET + 界面态优化
│   → 参考 D（hBN异质结）
│     680 cm²/Vs迁移率、>10⁸ ON/OFF、无表面受主
│     代价：需单晶金刚石衬底 + hBN单晶
│
└─ 颠覆性/长远探索 + 二维电子学
    → 关注 E（Diamane）
      全新"石墨烯→金刚石"范式、原子级厚度
      代价：以理论预测为主、实验验证极少
```

## 交叉比对关键结论

### 1. 界面质量 vs 工艺兼容性：不可兼得

- **B（直接键合）** 提供业界最优的亚纳米键合界面和最高质量的单晶膜，但需复杂的转移键合工艺
- **C（低温生长）** 以400°C的极致CMOS友好性换取略低于单晶的热/电性能
- **A（MPCVD）** 在大面积可制造性上最优，但880°C高温将应用限制在前道工艺

### 2. hBN范德华界面是跨路线共性启示

对标D的hBN/金刚石界面研究揭示了一个可跨路线复用的原则：**以无悬挂键的范德华界面替代传统氧化物/共价键界面**，可大幅降低界面态密度至 6.8×10¹¹ cm⁻² eV⁻¹。这一思路可反哺到A/B/C路线的界面钝化设计中。

### 3. 二维路线（Diamane）代表范式转移但远未成熟

对标E展示了"不从金刚石出发，而从石墨烯出发"的新逻辑。然而理论为主、实验极少（2018年首例），建议作为长期关注方向而非近期技术选型依据。

### 4. 成本-性能的Pareto前沿

```
性能 ↑
  │    B ★                    （高成本，高性能）
  │         D ★               （中等成本，FET专用）
  │     C ★                   （低成本，高兼容性）
  │   A ★                     （中等成本，大面积）
  │               E ★         （待定成本，早期）
  └──────────────────→ 成本
```

A/C 位于性价比最优区间；B 位于绝对性能最优区间；D/E 分别服务于特定场景和远期储备。
