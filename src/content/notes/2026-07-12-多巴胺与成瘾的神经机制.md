# 多巴胺与成瘾的神经机制

> 成瘾是「强化学习系统被劫持」——多巴胺奖励预测误差信号被药物或行为异常放大，导致动机转向失控。

---

## 多巴胺系统的正常功能

### 奖励预测误差

回顾 [[强化学习与多巴胺系统]] 中的核心发现：

```
实际奖励 > 预期 →  多巴胺 ↑（正预测误差）→ 学习「这个动作值得重复」
实际奖励 = 预期 →  多巴胺 →（无变化）   → 维持已有策略
实际奖励 < 预期 →  多巴胺 ↓（负预测误差）→ 学习「这个动作应该避免」
```

### 多巴胺的三条主要通路

| 通路 | 起点 → 终点 | 功能 |
|:----:|------------|------|
| **中脑边缘通路** | 腹侧被盖区 → 伏隔核 | 奖励、动机、欲望（「想要」） |
| **中脑皮层通路** | 腹侧被盖区 → 前额叶皮层 | 工作记忆、执行控制 |
| **黑质纹状体通路** | 黑质 → 背侧纹状体 | 运动控制（→ 帕金森病） |

**成瘾主要涉及中脑边缘通路。**

---

## 成瘾发生时发生了什么

### 阶段 1：正常学习

```
吃东西 → 多巴胺释放（+1）→ 强化「寻找食物」的行为
         下次预测到食物 → 多巴胺在预期时刻释放
         实际吃到 → 多巴胺不再额外释放（预测准确）
```

### 阶段 2：药物劫持

```
可卡因/安非他命：
  → 阻断多巴胺回收 → 突触间隙多巴胺浓度暴增 10-100 倍
  → 信号强度远远超过任何自然奖励
  → 大脑被「超自然信号」淹没

药物线索（看到注射器/白色粉末）：
  → 多巴胺在此刻大量释放
  → 比「实际获得奖励」本身的信号还强
```

### 阶段 3：动机转移

```yaml
随着时间推移：

前额叶皮层（执行控制）：
  ↓ 活动减弱
  → 无法抑制冲动

伏隔核（欲望/动机）：
  ↑ 对药物线索的反应增强
  → 药物线索自动吸引注意力

自然奖励（食物/社交/爱好）：
  ↓ 反应减弱（耐受）
  → 只有药物能带来满足感

最终：
  「不想要但需要」→ 成瘾的核心悖论
```

## 成瘾与强化学习的精确对应

```yaml
强化学习概念         神经对应           成瘾中的异常
─────────────      ──────────        ────────────
奖励函数            多巴胺基线          药物使奖励信号超出正常范围
策略学习            行为选择            策略偏向药物寻求
探索 vs 利用        新奇 vs 惯常        过度利用（重复用药），探索减少
折扣因子 γ          冲动性              高 γ 偏向即时奖励 → 冲动决策
预测误差            多巴胺瞬发反应      药物线索产生虚假的预测误差
行为习惯化          背侧纹状体          从有意使用 → 自动化的习惯
```

## 对 AI 安全的启发

```
成瘾研究给 AI 的一个重要警示：

如果强化学习代理的「奖励函数」设计不当，
它也会出现类似成瘾的行为：
  - 找到奖励信号的「漏洞」后反复利用
  - 忽略任务的实际目标，追逐代理奖励
  - 对其他目标「视而不见」

这就是「奖励黑客」（Reward Hacking）问题——
是当前 AI 安全研究的核心挑战之一。
```
