# 视觉皮层的层级处理（V1 → V2 → V4 → IT）

> 视觉皮层从枕叶到颞叶构成了一个层级处理系统——从简单边缘检测到复杂物体识别。这正是 CNN 架构的生物学原型。

## 处理层级

```
V1（初级视觉皮层）— 枕叶
  检测边缘、朝向、简单线条
   ↓
V2（次级视觉皮层）— 枕叶
  检测轮廓、纹理、立体深度
   ↓
V4 — 枕-颞叶交界
  检测颜色、复杂形状
   ↓
IT（下颞叶皮层）— 颞叶
  检测完整物体（面孔、手、动物等）
```

## 关键发现

- **Hubel & Wiesel（1959，诺贝尔奖）**：
  - 在猫的视觉皮层中发现「简单细胞」（对特定方向线条反应）和「复杂细胞」（对运动方向反应）
  - 这个发现直接启发了后来的 CNN 架构

## 与 CNN 的对应

| 大脑视觉系统 | CNN |
|------------|-----|
| V1 简单细胞（边缘检测） | 浅层卷积核 |
| V2（纹理/轮廓） | 中层卷积核 |
| V4 / IT（物体识别） | 深层全连接层 |
| 层级递增的感受野 | 池化层扩大感受野 |
