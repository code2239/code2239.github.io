## 1. 概念起源与核心开发者

### 1.1 思想溯源

世界模型的哲学根源可追溯到 **1943 年 Kenneth Craik** 提出的"内部模型"（Internal Model）理论——人类大脑之所以能快速应对复杂世界，是因为我们在脑海中内置了一个能模拟现实的小型"机器"。

**1991 年**，Rich Sutton 提出 **Dyna 架构**，首次将"世界模型"确立为强化学习智能体的内部基础能力。

### 1.2 现代奠基

| 人物 | 贡献 |
|------|------|
| **David Ha & Jürgen Schmidhuber**（Google Brain）| 2018 年论文《World Models》正式确立名称，提出 **V（视觉）+ M（记忆/预测）+ C（控制）** 三模块框架 |
| **Yann LeCun**（Meta / AMI Labs）| 提出 JEPA 架构，2026 年创立 AMI Labs 获 $10 亿+ 融资 |
| **李飞飞**（World Labs）| 空间智能路线，创立 World Labs（估值 $50 亿）|
| **黄仁勋**（NVIDIA）| Cosmos 平台，推动工业级世界模型开源 |
| **王仲远**（智源研究院）| 提出 NSP（Next-State Prediction）新范式 |

### 1.3 当前核心参与机构（2026）

| 阵营 | 代表机构 | 技术路线 |
|------|---------|---------|
| **产业巨头** | NVIDIA、Google DeepMind、OpenAI、Meta、阿里、腾讯、华为 | 全栈布局，自研+开源 |
| **学术机构** | 智源研究院、上海AI Lab、中科院、MIT、Stanford、UC Berkeley | 前沿理论、系统综述 |
| **创业公司** | AMI Labs（LeCun）、World Labs（李飞飞）、大晓机器人（开悟）、Momenta | 专注突破 |

## 2. 底层技术架构

### 2.1 统一定义（2026 年学术共识）

> **世界模型**是一个**动作条件化的预测系统**（Action-conditioned Predictive System），在有限计算资源下学习环境的结构与动力学，使智能体能够在内部表征中进行预测、规划和推理。——上海AI Lab、中科院、NTU 联合定义

### 2.2 三大核心功能

```
┌──────────────────────────────────────────────┐
│                世界模型（World Model）          │
├───────────────────┬────────────────┬─────────┤
│  📡 Simulator     │  🎨 Renderer   │  🧠 Planner  │
│  （模拟器）        │  （渲染器）      │  （规划器）   │
│  预测状态如何演化  │  将状态转为可视  │  反事实推演   │
│                   │  输出（视频/图） │  选择最优动作 │
└───────────────────┴────────────────┴──────────┘
```

### 2.3 五大技术流派（2026 年全景）

#### 流派一：视觉与像素生成（经验主义）
| 子路线 | 代表 | 核心思想 | 硬伤 |
|--------|------|---------|------|
| **扩散视频模型** | Sora、Cosmos-Predict、阿里 Wan | 视频生成=世界模拟，暴力压缩"顿悟"物理 | 物理一致性弱、长程"相位腐败" |
| **自回归交互模型** | Genie 3、IRIS、Wayve Gaia-1 | 下一帧预测=Next-token，显式输入动作 | 暴露偏差导致误差指数发散 |

#### 流派二：认知与潜空间（理性主义）
| 子路线 | 代表 | 核心思想 | 硬伤 |
|--------|------|---------|------|
| **JEPA 架构** | V-JEPA 2.1、AMI Labs | 扔掉像素，在高维潜空间预测特征变化 | 丢失细粒度视觉动态 |
| **RL 潜空间模型** | DreamerV3、MuZero、SimPLe | "精神游戏厅"——潜空间中预演未来 | Sim-to-Real Gap |

### 2.4 2026 年架构共识：分层混合

行业已放弃"单模型包揽一切"，走向 **Reasoner（推理器）+ Generator（生成器）** 的前后端分离架构：

```
输入（文本/图像/视频/动作）
         │
         ▼
┌────────────────────────────┐
│  Reasoner Tower（推理器）    │  ← JEPA / RL 潜空间思想
│  处理物理隐状态推演          │
│  输出：结构化场景理解        │
└──────────┬─────────────────┘
           │ conditioning
           ▼
┌────────────────────────────┐
│  Generator Tower（生成器）   │  ← 扩散模型 / 3DGS
│  渲染最终视觉效果           │
│  输出：视频 / 图像 / 动作    │
└────────────────────────────┘
```

典型代表：[NVIDIA Cosmos 3](https://developer.nvidia.com/blog/develop-physical-ai-reasoning-world-and-action-models-with-nvidia-cosmos-3/) · [智源悟界·Orca](https://hub.baai.ac.cn/view/56211)

## 4. 功能全景解析

### 主要功能（一句话概括）
> 世界模型是 AI 理解、预测和干预物理世界的核心能力——从"预测下一个词"跃迁到"预测下一个物理状态"。

### 核心能力

#### 4.1 🌍 物理世界理解（World Understanding）
- 解析物体交互、运动轨迹、因果关系
- 理解重力、碰撞、摩擦等物理约束
- **关键技术**：Reasoner VLM、JEPA 潜空间编码

#### 4.2 🔮 未来状态预测（World Prediction）
- 从当前状态预测下一个物理状态
- 支持从文本/图像/视频/动作多种条件输入
- **关键技术**：NSP 范式、扩散 Transformer、自回归预测
- 代表性基准：[WorldModelBench](https://hub.baai.ac.cn/view/56227) · [Physics-IQ](https://developer.nvidia.com/blog/develop-physical-ai-reasoning-world-and-action-models-with-nvidia-cosmos-3/)

#### 4.3 🎬 可控视频生成（Controllable Video Generation）
- 生成符合物理规律的视频
- 支持多种控制信号：分割图、深度图、边缘图、3D 边界框、LiDAR
- **关键技术**：Multi-ControlNet、引导式扩散
- 代表：[NVIDIA Cosmos Transfer](https://github.com/nvidia-cosmos/cosmos-transfer2.5) · [阿里 Wan](https://www.36kr.com/p/3868221222851589)

#### 4.4 🤖 动作策略生成（Action Policy）
- 将世界理解直接转化为可执行动作
- 支持前向动力学、逆向动力学、策略生成
- **关键技术**：World Action Model（WAM）、Action Tokenization
- 代表：[Cosmos Policy DROID](https://developer.nvidia.com/blog/develop-physical-ai-reasoning-world-and-action-models-with-nvidia-cosmos-3/) · [Qwen-RobotWorld](https://qwen.ai/blog?id=qwenvla)

#### 4.5 🔄 合成数据生成（Synthetic Data Engine）
- 为机器人/自动驾驶生成无限多样化训练数据
- 覆盖长尾边缘场景，大幅降低真实数据采集成本
- **关键技术**：数据飞轮（Data Flywheel）、模拟器闭环
- 代表：[蔚来 NWM 2.0](https://www.kucoin.com/news/flash/chinese-tech-giants-and-automakers-expand-world-model-capabilities-in-2026)（AEB 覆盖 6.7 倍） · [小鹏 X-World](https://www.kucoin.com/news/flash/chinese-tech-giants-and-automakers-expand-world-model-capabilities-in-2026)（日仿真 3000 万公里）

#### 4.6 🧪 反事实推理与规划（Counterfactual Reasoning）
- 在内部模拟"如果...会怎样？"
- 支持多分支探索和最优路径选择
- **关键技术**：潜空间 Rollout、树搜索、MCTS

> **信息源**：[arXiv 2606.00133](https://export.arxiv.org/abs/2606.00133) · [上海AI Lab 综述](https://hub.baai.ac.cn/view/56227) · [中科院/牛津 综述](http://mp.weixin.qq.com/s?__biz=MzU2ODgzMTM5NA==&mid=2247504820&idx=2&sn=4fdc542074c3f1adf361fdae917a5bdb) · [36氪 大厂版图](https://www.36kr.com/p/3868221222851589)

## 5. 功能实现案例

### 案例：蔚来 NWM 2.0 —— 世界模型驱动的自动驾驶

**场景**：车辆在高速行驶中，前方突然出现静止障碍物（如抛锚车辆），系统需要在极短时间内判断刹车距离、避让路径和乘员安全。

**传统方案**：依赖预先标注的"障碍物"数据集 + 规则触发AEB。覆盖场景有限，长尾情况（异形车辆、特殊光照）容易漏判。

**世界模型方案（蔚来 NWM 2.0）：**

```
1. 实时感知
   ↓ 7 路环视摄像头输入
2. 世界模型推演
   ↓ 将当前场景抽象为物理状态 → 预测未来 3 秒内所有可能的演变路径
   ├─ 路径 A：紧急刹车 → 预测刹停距离 + 后方追尾风险
   ├─ 路径 B：向左避让 → 预测相邻车道车辆位置变化
   └─ 路径 C：轻微转向减速 → 预测最佳平衡点
3. 规划器选择最优路径
   ↓
4. 直接输出方向盘/踏板原始信号
```

**结果**：
- AEB 覆盖场景为传统方案的 **6.7 倍**
- 2026 年 6 月推送，覆盖 **70 万+ 用户**
- 首次国内实现智驾模型直接输出方向盘/踏板原始信号（端到端）

**来源**：[KuCoin 报道](https://www.kucoin.com/news/flash/chinese-tech-giants-and-automakers-expand-world-model-capabilities-in-2026)
