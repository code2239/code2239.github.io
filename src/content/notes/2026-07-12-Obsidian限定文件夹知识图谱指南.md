# Obsidian 限定文件夹知识图谱指南

> 如何将 Obsidian 的图形视图（Graph View）范围限制到**某一个文件夹**的 Markdown 文件。

---

## 1. 全局图形视图 + 路径过滤器

**适用场景**：想查看某个文件夹（含子文件夹）内所有笔记之间的链接关系。

### 操作步骤

1. 点击左侧栏的 **圆形图标** 打开全局图形视图（或快捷键 `Cmd/Ctrl + Shift+G`）
2. 在图形窗口顶部的 **搜索过滤框**（Labels: 输入...）中输入：

```
path:你要的文件夹路径
```

### 常用过滤示例

| 输入 | 效果 |
|------|------|
| `path:30.areas` | 显示 `30.areas` 文件夹及其所有子文件夹中的笔记 |
| `path:"30.areas/finance"` | 只显示 `finance` 子文件夹（注意路径用引号包裹） |
| `path:notes` | 显示 `notes/` 下的所有笔记 |

### 进阶组合过滤

```bash
# 多个文件夹取交集
path:30.areas path:daily

# 排除某个文件夹
-path:"30.areas/archive"

# 文件夹 + 标签组合
path:projects -tag:done

# 文件夹 + 链接关系
path:30.areas -regex:".*\.png$"   # 排除图片文件
```

---

## 2. 局部图谱（Local Graph）

**适用场景**：聚焦某篇笔记周围的关联网络，而非整个文件夹。

### 操作步骤

1. 打开目标文件夹中的任意一篇笔记
2. 快捷键 `Cmd/Ctrl + G` 或命令面板搜索「打开局部图谱」
3. 点击图谱窗口右上角的 **⚙️ 齿轮图标**，进入设置
4. 调整参数：

| 参数 | 推荐值 | 说明 |
|------|:------:|------|
| 深度（Depth） | `1` 或 `2` | 向外扩展几层链接关系 |
| 仅显示现有文件 | ✅ 勾选 | 隐藏断裂链接（指向不存在文件的链接）|
| 根据链接方向着色 | 按需 | 区分出链与入链 |

### 局部图谱 vs 全局过滤对比

| 维度 | 全局图谱+过滤 | 局部图谱 |
|------|-------------|---------|
| 范围 | 文件夹内**所有笔记** | 以某篇笔记为中心的**辐射网络** |
| 最佳用途 | 俯瞰文件夹结构 | 单篇笔记的上下文探索 |
| 动态更新 | 需手动刷新过滤 | 切换笔记自动更新 |

---

## 3. 保存查询图谱（适合常查看的文件夹）

如果你经常检查同一个文件夹的图谱，可以将其保存为预设。

### 操作步骤

1. 在全局图形视图中输入 `path:你的文件夹` 等过滤条件
2. 点击右上角 **三个点**（更多选项）→ **保存图形过滤器**
3. 输入名称（如「学习笔记」「项目图谱」）
4. 保存后，可从「已保存的图形过滤器」下拉菜单中快速切换

---

## 4. CSS 定制：文件夹着色（进阶）

如果你想让不同文件夹的笔记在图形中显示不同颜色，可以在 `obsidian.css` 或 ` snippets/` 中添加以下样式（需要已安装并开启「样式设置」插件）：

```css
/* 给 30.areas 文件夹的节点设置颜色 */
.graph-view.color-fill-tag {
  /* 默认标签颜色 */
}

/* 通过 data-path 属性筛选 —— 需配合 Obsidian 内部类名 */
```

但更简单的方式是：在图形设置中直接用**分组（Groups）**功能：

1. 图形视图 → 右上角 ⚙️ → **分组（Groups）**
2. 添加新分组，输入 `path:30.areas`
3. 为该分组指定颜色
4. 该文件夹下的所有节点将以该颜色显示，其他节点保持默认

## 5. 插件增强方案

| 插件名称 | 功能 | 推荐理由 |
|---------|------|---------|
| **[Graph Analysis](obsidian://show-plugin?id=graph-analysis)** | 超强图谱分析工具 | 支持按文件夹/标签/链接类型精细筛选，比原生强大得多 |
| **[ExcaliBrain](obsidian://show-plugin?id=excalibrain)** | 思维导图式图谱 | 更直观的层级展示，支持文件夹筛选 |
| **[Obsidian Git](obsidian://show-plugin?id=obsidian-git)** | 配合图谱管理 | 查看文件夹变更历史，辅助版本化知识管理 |

---

## 6. 总结对比

| 需求 | 最佳方案 |
|------|---------|
| 只看某个文件夹的全部笔记关系 | `path:文件夹路径` 过滤 |
| 看某篇笔记相邻的网络 | 局部图谱 + 调整深度 |
| 经常切换查看不同文件夹 | 保存多个图形过滤器 |
| 区分不同文件夹的节点颜色 | 图形设置 → 分组配色 |
| 深度分析和筛选 | Graph Analysis 插件 |
