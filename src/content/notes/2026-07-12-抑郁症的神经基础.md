# 抑郁症的神经基础

> 抑郁症不是「想开点就能好」的情绪问题——它是涉及前额叶、杏仁核、海马体等多个脑区的**神经网络疾病**。

---

## 核心症状与对应的脑区异常

| 症状 | 涉及的脑区 | 异常模式 |
|:----:|-----------|:--------:|
| **持续情绪低落** | 前额叶皮层 + 杏仁核 | 前额叶对杏仁核的抑制作用减弱 |
| **快感缺失（对什么都没兴趣）** | 伏隔核（奖励系统） | 多巴胺信号减弱 |
| **认知功能下降（注意力/记忆）** | 前额叶皮层 | 活动降低，前额叶萎缩 |
| **负向思维反刍** | 默认模式网络 (DMN) | DMN 过度活跃 + 无法正常切换 |
| **睡眠/食欲改变** | 下丘脑 | 昼夜节律失调 |
| **海马体萎缩** | 海马体 | 长期应激 → 糖皮质激素 → 神经元死亡 |

## 神经化学层面的主要理论

### 单胺假说（经典理论）

```yaml
核心想法：
  抑郁 = 单胺类神经递质（血清素、去甲肾上腺素、多巴胺）不足

支持证据：
  SSRI（百忧解等）通过增加突触间隙血清素改善症状
  → 所以「血清素不足导致抑郁」

局限：
  ① SSRI 需要 2-4 周才起效（而药物本身数小时就改变血清素水平）
  ② 约 1/3 的患者对 SSRI 无反应
  ③ 降低正常人的血清素水平 → 并不会让他们抑郁

→ 单胺假说「不完整但没有错」——它只是故事的一部分
```

### 应激-皮质醇假说

```
长期压力 → HPA 轴过度活跃 → 皮质醇持续偏高
  → 皮质醇对海马体有神经毒性
  → 海马体萎缩
  → 记忆受损 + 情绪调节能力下降
  → 对压力的感知更加负面 → 恶性循环

→ 解释了为什么慢性压力是抑郁的最强预测因子之一
→ 也解释了为什么抑郁患者的海马体体积缩小
```

### 神经可塑性假说（当前主流）

```
不是「化学失衡」，而是「连接失衡」：

抑郁 = 神经可塑性下降 + 连接模式固定化
  - 正反馈回路被卡在「负面」模式
  - 大脑无法通过正常的学习和适应「离开」这个模式
  
解释了很多现象：
  - 为什么 SSRI 需要 2-4 周起效（不是改变递质水平，而是触发可塑性连锁反应）
  - 为什么运动对抑郁有效（运动促进 BDNF 释放 → 海马体神经发生）
  - 为什么认知行为疗法有效（通过改变思维模式重塑连接）
```

## 三个网络的功能异常

回顾 [[大脑脑区与功能全景图#全脑功能网络]]：

| 网络 | 抑郁中的异常 | 表现 |
|:----:|:-----------:|------|
| **默认模式网络 (DMN)** | **过度活跃**，且与 CEN 的切换减弱 | 反刍（反复想同一件负面的事）、无法「走神」 |
| **中央执行网络 (CEN)** | **活动降低** | 注意力差、决策困难、执行功能下降 |
| **突显网络 (SN)** | 对负面信息**过度敏感** | 对微不足道的负面事件反应过度 |
