# 强化学习与多巴胺系统

> 强化学习的核心算法——时序差分学习（TD Learning）——直接受大脑多巴胺神经元的「奖励预测误差」信号启发。

## 关键发现：多巴胺 = 奖励预测误差

```
Wolfram Schultz（1990年代）在猴子实验中：

实验 1：猴子意外得到果汁
  → 多巴胺神经元强烈放电（正预测误差）

实验 2：猴子学会「听到声音 → 得到果汁」
  → 听到声音时放电，得到果汁时不再放电（预测准确，误差为零）

实验 3：猴子听到声音 → 以为要得果汁 → 但没给
  → 在「本该得果汁」的时刻放电显著受抑制（负预测误差）
```

**这正是强化学习中 TD 误差的神经对应**：

```yaml
TD 误差 = 实际奖励 + 下一状态价值 − 当前状态价值

多巴胺信号 = 实际奖励 + 下一状态价值 − 当前状态价值

两者完全一致。
```

## 对 AI 的启发

- **Sutton & Barto（1988，1998）**：提出 TD 学习算法，奠基现代强化学习
- **DeepMind DQN（2013）**：将深度学习 + 强化学习结合，在 ATARI 游戏中超越人类
- **AlphaGo / AlphaZero（2016+）**：深度强化学习 + 蒙特卡洛树搜索 → 击败围棋世界冠军

## AI 与大脑强化学习的差异

| 维度 | 生物多巴胺系统 | 人工强化学习 |
|:----:|--------------|:-----------:|
| 计算速度 | 毫秒级（突触传递） | 纳秒级（GPU） |
| 学习样本 | 在线学习，实时更新 | 离线的经验回放缓冲 |
| 探索策略 | 多巴胺基底节环路内置的探索机制 | epsilon-greedy 或更复杂的探索算法 |
| 多任务 | 单一系统处理所有类型的奖励 | 每个任务的奖励函数需单独设计 |
