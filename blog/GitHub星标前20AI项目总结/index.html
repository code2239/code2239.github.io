🌟 GitHub 星标前 20 的 AI 相关开源项目

调研日期：2026-07-05
排序依据：GitHub Star 数量（星标数为近似值，综合自多个排行榜）

1. TensorFlow — tensorflow/tensorflow

开发者：Google Brain / Google DeepMind（美国） ｜ ⭐ ~194K
GitHub 链接

Google 出品的工业级深度学习框架，支持大规模分布式训练和跨平台部署（TF Serving / TF Lite / TF.js / TFX）。在移动端/边缘设备和 TPU 推理方面仍是最优选择，学术研究中被 PyTorch 超越。

2. n8n — n8n-io/n8n

开发者：n8n GmbH（德国柏林）｜ ⭐ ~184K
GitHub 链接

开源工作流自动化平台，2025–2026 年原生集成 AI 节点（LangChain / OpenAI / Claude）。可视化拖拽编排 + 400+ 集成，适合从简单任务到复杂 AI 管线的自动化编排。

3. AutoGPT — Significant-Gravitas/AutoGPT

开发者：Significant Gravitas（社区驱动）｜ ⭐ ~183K
GitHub 链接

自主 AI Agent 的先驱项目。给定目标后自动分解任务、调用工具、浏览网页、执行代码，逐步完成目标。首次让”AI 自主干活”的概念走向大众。

4. Ollama — ollama/ollama

开发者：Ollama Inc.（美国）｜ ⭐ ~169K
GitHub 链接

本地运行大模型的瑞士军刀。一行命令即可下载运行 DeepSeek、Llama、Qwen、Gemma、Mistral 等主流模型，提供 OpenAI 兼容 API。推动了”本地 AI”运动的普及。

5. Stable Diffusion Web UI — AUTOMATIC1111/stable-diffusion-webui

开发者：AUTOMATIC1111（个人开发者，社区维护）｜ ⭐ ~162K
GitHub 链接

Stable Diffusion 最完整的 Web 交互界面，支持文生图、图生图、ControlNet、LoRA 等。Gradio 构建，插件生态丰富，是 AI 绘画领域装机量最大的开源项目。

6. Hugging Face Transformers — huggingface/transformers

开发者：Hugging Face（美国/法国）｜ ⭐ ~159K
GitHub 链接

统一 API 调用 200+ 模型家族（BERT、GPT、LLaMA、DeepSeek 等），支持 PyTorch / TensorFlow / JAX。Hub 托管 200 万+ 模型和 50 万+ 数据集，是 ML 领域的通用接口层。

7. Dify — langgenius/dify

开发者：LangGenius（中国广州/深圳，创始人张路宇）｜ ⭐ ~147K
GitHub 链接

低代码 LLM 应用开发平台，内建 RAG 知识库、Agent 工作流、对话管理、模型监控。支持接入主流 API 和本地模型，从原型到生产部署的一站式方案。

8. LangChain — langchain-ai/langchain

开发者：LangChain Inc.（美国，创始人 Harrison Chase）｜ ⭐ ~131K
GitHub 链接

LLM 应用框架，提供链式调用、Agent 编排、RAG、记忆管理等核心模块。2026 年生态扩展至 LangGraph（状态工作流）和 LangSmith（调试监控），是 LLM 生态中的”操作系统层”。

9. Open WebUI — open-webui/open-webui

开发者：Timothy J. Baek 及社区｜ ⭐ ~132K
GitHub 链接

自托管 AI 聊天界面，支持多模型切换、RAG 知识库、网页搜索、语音输入、权限管理。2026 年是本地部署 AI 助手的事实标准前端，与 Ollama 最常配合使用。

10. ComfyUI — comfyanonymous/ComfyUI

开发者：Comfyanonymous（个人开发者）｜ ⭐ ~109K
GitHub 链接

基于节点图的 Stable Diffusion 工作流编辑器，拖拽连接各处理节点构建图像生成管线，可精确控制每一步参数。专业 AI 画师首选工具。

11. llama.cpp — ggerganov/llama.cpp

开发者：Georgi Gerganov（保加利亚）｜ ⭐ ~103K
GitHub 链接

纯 C/C++ LLM 推理引擎，无需 GPU 即可在 CPU 高效运行大模型。量化技术大幅降低内存占用，支持 Metal / CUDA / Vulkan 加速。“人人可本地跑大模型”的底层基石。

12. PyTorch — pytorch/pytorch

开发者：Meta（Facebook）→ PyTorch Foundation（Linux 基金会）｜ ⭐ ~99K
GitHub 链接

动态计算图深度学习框架，2026 年 ~80%+ arXiv ML 论文使用。Hugging Face 默认后端，研究社区事实标准。支持 GPU 加速、分布式训练、TorchScript 导出。

13. OpenAI Whisper — openai/whisper

开发者：OpenAI（美国旧金山）｜ ⭐ ~97K
GitHub 链接

开源语音识别模型，支持 99+ 语言的转写和翻译。基于大规模弱监督训练的 Encoder-Decoder，鲁棒性强。广泛用于会议转录、字幕生成和语音助手。

14. OpenCV — opencv/opencv

开发者：Intel 发起，社区维护｜ ⭐ ~87K
GitHub 链接

计算机视觉领域最广泛的开源库，2500+ 优化算法（图像处理、检测、跟踪、识别）。支持 C++/Python/JS 多语言接口，2025–2026 年增强了对 ONNX/TensorFlow 的直接推理。

15. GPT4All — nomic-ai/gpt4all

开发者：Nomic AI（美国）｜ ⭐ ~77K
GitHub 链接

消费级 CPU 即可运行的本地 LLM 聊天机器人。提供跨平台桌面客户端，支持本地 RAG 文档检索。核心理念：AI 完全在本地运行，保护隐私，无需联网。

16. vLLM — vllm-project/vllm

开发者：UC Berkeley → vLLM 社区｜ ⭐ ~76K
GitHub 链接

高吞吐 LLM 推理引擎，PagedAttention 技术将显存利用率提升 2–4 倍，支持 Continuous Batching / Prefix Caching。已成为 LLM 生产部署的事实标准。

17. GPT Academic — binary-husky/gpt_academic

开发者：Binary Husky（中国）｜ ⭐ ~70K
GitHub 链接

专为学术设计的 LLM 交互工具：论文翻译/润色/总结、LaTeX 解析、代码项目分析、自定义函数插件。支持同时调用 GPT、Claude、GLM、DeepSeek 等多模型对比。

18. MetaGPT — geekan/MetaGPT

开发者：DeepWisdom（中国，创始人吴承霖）｜ ⭐ ~68K
GitHub 链接

多 Agent 协作框架，模拟软件公司角色（PM/架构师/工程师/QA）。一句话需求 → 多个 AI 角色自动协作输出 PRD、设计文档、代码和测试。体现多智能体协作前沿理念。

19. Segment Anything — facebookresearch/segment-anything

开发者：Meta Research（美国）｜ ⭐ ~54K
GitHub 链接

通用图像分割模型 SAM，零样本分割任意图像中的任意对象（点选/框选即可）。2026 年 SAM 2 支持视频跟踪分割，广泛用于医学影像、自动驾驶和创意设计。

20. YOLOv5 — ultralytics/yolov5

开发者：Ultralytics（美国，创始人 Glenn Jocher）｜ ⭐ ~57K
GitHub 链接

实时目标检测标杆模型，速度快精度高。PyTorch 实现，支持导出到 ONNX / TFLite / CoreML。广泛应用于安防、自动驾驶、工业质检。Ultralytics 已进化到 YOLOv11，但 YOLOv5 因稳定性仍是装机量最大。
