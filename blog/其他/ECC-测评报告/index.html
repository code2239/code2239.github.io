4. 功能全景解析

4.1 主要功能

一个教育性质的 ECC 双重实现（Java + Python），覆盖椭圆曲线密码学的加密、解密及数字签名三大核心操作，面向密码学初学者和高校课程学习者。[2][3]

4.2 核心功能与关键技术

核心功能描述关键技术
1. 椭圆曲线密钥对生成基于有限域 ( \mathbb{F}_p ) 上椭圆曲线点群运算，从随机私钥推导公钥。标量乘法；Java 端可能调用 Bouncy Castle 或 java.security；Python 端可能使用 ecdsa、cryptography 或纯 Python 实现 [5]
2. ECDSA 数字签名与验证椭圆曲线数字签名算法，包括签名生成（利用随机 nonce）和公钥验证两个子流程。ECDSA 是 ECC 在区块链（如 Bitcoin）、TLS 1.3 等场景中最核心的应用 [4][6]
3. ECDH 密钥交换椭圆曲线 Diffie-Hellman 协议，使双方在不安全的信道上协商共享密钥。基于椭圆曲线上的点乘运算实现安全的密钥协商 [6]
