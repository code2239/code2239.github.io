🤖 GitHub 开源 AI 功能全景报告

调研日期：2026-07-05
说明：总结当前 GitHub 上 AI 能够实现的十大类核心功能，每类列出星标前 5 的代表性工具及其 GitHub 链接。

能力说明：在本地或服务器部署、运行、量化大语言模型，无需依赖商业 API。让任何开发者都能在自有硬件上跑 LLM。

排名工具⭐ Stars简介GitHub

4GPT4All~77K消费级 CPU 即可运行的 LLM 桌面客户端，零 GPU 依赖nomic-ai/gpt4all
5LocalAI~30K本地运行 LLM/图像/音频模型的自托管替代 OpenAI APImudler/LocalAI
